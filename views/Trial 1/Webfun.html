This module will help you to better understand the appearances of tuberculosis on chest radiographs.

Five-second Frenzy

In this mode each radiograph is displayed for a maximum of five seconds, you will have a chance to select your findings before moving on to the next radiograph

Timed Mode

In this mode the time it takes to complete each radiograph is monitored, but there is no limit to the length of time spent on any item.

Quiz Mode

In this mode answers to each radiograph are selected, but feedback is not displayed until all items are completed.

Tutor Mode

In this mode after you have selected an answer, immediate feedback is provided.
